Other Tools

  1. ChainIDE - browser code editor to deploy and interact with smart contracts. Documentation on Godwoken usage.
  2. Gnosis Safe: deployed and verified smart contracts to deterministic addresses (equivalent to other EVM chains) - https://github.com/safe-global/safe-deployments/pull/94. Official support hasn't been added yet by Gnosis Safe team.
  3. Multicall3: deployed and verified smart contract to deterministic address (equivalent to other EVM chains) - https://gw-explorer.nervosdao.community/address/0xcA11bde05977b3631167028862bE2a173976CA11/contracts.
  4. nervos-godwoken-integration: a library to integrate Layer 1 and Layer 2 Nervos specific actions such as account creation on Layer 2.
